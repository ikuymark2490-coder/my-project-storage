Notifications

01. แจ้งเตือนแบบพื้นฐาน

showNotify("ภารกิจสำเร็จ!");
อธิบาย: ใช้ฟังก์ชัน showNotify() เพื่อแสดงกล่องข้อความแจ้งเตือนที่ด้านบนของหน้าจอชั่วคราว เหมาะสำหรับแจ้งสถานะทั่วไป

02. แจ้งเตือนพร้อมสั่นเครื่อง

showNotify("⚠️ พลังงานต่ำ!");
navigator.vibrate(200);
อธิบาย: navigator.vibrate(200) สั่งให้โทรศัพท์สั่นเป็นเวลา 200 มิลลิวินาที (ใช้ได้เฉพาะบน Android/มือถือที่รองรับ) เพื่อเน้นย้ำความสำคัญ

LOGIC & INTERACTION

03. ระบบสลับสถานะ (Toggle Visual)

if (this.style.opacity == "0.5") {
    this.style.opacity = "1";
} else {
    this.style.opacity = "0.5";
}
อธิบาย: ตรวจสอบว่าวัตถุ (this) จางอยู่หรือไม่ ถ้าจางให้กลับมาเข้ม ถ้าเข้มให้ทำให้จาง ใช้สำหรับปุ่มกดที่ต้องการแสดงสถานะ เปิด/ปิด

04. ระบบสุ่มเลขแบบกำหนดขอบเขต

let rand = Math.floor(Math.random() * 10) + 1;
showNotify("คุณสุ่มได้เลข: " + rand);
อธิบาย: Math.random() * 10 สุ่มเลข 0-9 จากนั้น Math.floor จะปัดเศษทิ้ง และ +1 เพื่อให้ผลลัพธ์เป็นเลข 1 ถึง 10

AUDIO ENGINE

05. เล่นเสียงแบบคุมระดับเสียง

let snd = new Audio("click.mp3");
snd.volume = 0.3;
snd.play();
อธิบาย: สร้างตัวแปรเสียงขึ้นมา แล้วกำหนด volume เป็น 0.3 (30%) เพื่อไม่ให้เสียงดังเกินไปก่อนสั่งให้เล่น (play)

STORAGE SYSTEM

06. ระบบเช็คข้อมูลเก่า (Auto Load)

let savedName = localStorage.getItem("player_name");
if(savedName) {
    showNotify("ยินดีต้อนรับกลับคุณ " + savedName);
} else {
    showNotify("ไม่พบข้อมูลผู้เล่น");
}
อธิบาย: พยายามดึงข้อมูลชื่อจากความจำเครื่อง (localStorage) ถ้ามีข้อมูล (if) ให้ทักทาย ถ้าไม่มี (else) ให้แจ้งว่าไม่พบข้อมูล

GAMEPLAY LOGIC

07. ระบบสุ่มดวง (Gacha/Chance)

let rate = Math.random() * 100; // สุ่ม 0-100
if (rate <= 5) {
    showNotify("⭐ ระดับตำนาน! (โอกาส 5%)");
} else if (rate <= 20) {
    showNotify("🔷 ระดับแรร์ (โอกาส 15%)");
} else {
    showNotify("⚪ ระดับทั่วไป");
}
อธิบาย: ใช้ Math.random() เพื่อสร้างตัวเลขทศนิยม แล้วคูณ 100 เพื่อทำเป็นเปอร์เซ็นต์ จากนั้นใช้ if-else เช็คช่วงตัวเลขเพื่อกำหนดระดับความหายาก

08. ระบบคูลดาวน์ปุ่ม (Prevent Spam)

if (window.isAction) return; // ถ้ากำลังทำงานอยู่ให้หยุด

window.isAction = true;
this.style.filter = "grayscale(100%)"; // ทำปุ่มสีเทา
showNotify("รอ 2 วินาที...");

setTimeout(() => {
    window.isAction = false;
    this.style.filter = "none"; // คืนสีปุ่ม
    showNotify("กดได้แล้ว!");
}, 2000);
อธิบาย: ใช้ตัวแปร window.isAction เป็นตัวล็อก (Flag) เพื่อไม่ให้คำสั่งทำงานซ้อนกันขณะที่ยังติดคูลดาวน์อยู่

LAYER MANIPULATION

09. สั่งงานวัตถุอื่นด้วย ID

let target = document.getElementById("el_12345");
target.style.display = "none"; // สั่งปิดเลเยอร์นั้น
showNotify("ปิดเลเยอร์เป้าหมายแล้ว");
อธิบาย: ใช้ document.getElementById เพื่อเข้าถึงวัตถุอื่นที่ไม่ใช่ตัวมันเอง (ต้องเอา ID มาจากหน้าจัดการ Layer ใน Engine)

10. เปลี่ยนข้อความในเลเยอร์

this.innerText = "กำลังโหลด...";
setTimeout(() => {
    this.innerText = "เสร็จสิ้น!";
}, 1000);
อธิบาย: innerText ใช้สำหรับเปลี่ยนตัวอักษรที่แสดงบนเลเยอร์นั้นๆ ทันที

STORAGE & DATA

11. ระบบสะสมแต้ม (Persistent Score)

let currentScore = Number(localStorage.getItem("score")) || 0;
currentScore += 10;
localStorage.setItem("score", currentScore);

this.innerText = "แต้มสะสม: " + currentScore;
showNotify("+10 แต้มบันทึกแล้ว!");
อธิบาย: ดึงค่าเก่ามาแปลงเป็นตัวเลข (Number) ถ้าไม่มีให้เริ่มที่ 0 จากนั้นบวกเพิ่มแล้วเซฟทับกลับไป ค่านี้จะคงอยู่แม้ปิดแอป

12. ล้างข้อมูลทั้งหมด (Reset Game)

if (confirm("ต้องการลบข้อมูลทั้งหมดใช่ไหม?")) {
    localStorage.clear();
    showNotify("ล้างข้อมูลสำเร็จ! กรุณารีเฟรช");
}
อธิบาย: confirm() จะเด้งหน้าต่างถามผู้ใช้เพื่อป้องกันการกดพลาด ถ้ากดยืนยันจะใช้ localStorage.clear() เพื่อลบทุกอย่างที่เคยเซฟไว้
